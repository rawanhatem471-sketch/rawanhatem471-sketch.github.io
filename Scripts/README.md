# Scripts

## Script 1

**الأم:** قومي ذاكري، الامتحان قرب.

**البنت:** مش هعرف أذاكر.

**الأم:** ليه؟

**البنت:** مشربتش قهوة.

الام تطلب من Breadfast.

**الأم:** طلبتلك بريدفاست

**البنت:** مش هيلحق.

وفجأة صوت الجرس الدليفري يوصل.

**البنت:** إيه ده؟! بسرعة كده؟

الديليفري يديها القهوه.

**البنت:** طب البرفيوم بتاعي خلص يعني انت عارفه اكيد مش هنزل كدخ

الجليفري بيدلها برفيوم من اللوكال يراندز في بريدفاست.

**البنت:**ليب جلوس!

*الدليفري بيديهولها

الموضوع بيتكرر مع كذا حاجه البنت بتتصدم بتياس بتكمل مذاكره وبتقوم عشان تنزل.


**Breadfast. سريع؟ آه. بس مش للفطار بس.**
